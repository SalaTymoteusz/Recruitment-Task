# Recruitment-Task-Codeaddict
Recruitment task for iOS Developer

TECHNOLOGIES:
- SWIFT
- MVC
- NAVIGATION WITH COORDINATOR PATTERN
- LAYOUTS CREATED IN CODE (UIKIT)
- VIEWCONTROLLERS STACK WITH UINAVIGATIONCONTROLLER
